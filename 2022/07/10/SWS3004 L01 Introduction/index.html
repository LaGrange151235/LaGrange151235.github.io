发表于 2022-07-10 分类于 NUS Summer Workshop 2022

Three Eras of Computing

  • Tabulating era 机械时代
  • Programming era 程序时代
  • Cognitive era 认知时代

Cognitive Computing

  • cognition: act of thinking
  • cognitive computing: simulating the act of thinking imitating human
  • explosion of (unstructured) data（非结构化数据） leads to the development of cognitive systems
  • discovery, engagement and decision（发现，研究， 决策）

Cognitive Computing Model


Cloud computing provides a platform for executing and delivering cognitive services to users.

Cloud Computing

Key Terms

  • Elastic resource（弹性资源）
  • Capacity planning（性能规划）
  • Scaling（扩展）
  • Cloud-based IT resources（基于云的IT资源）
  • Cloud service（基于云的IT资源）
  • Trust boundary（可信边界）

Definitions

  • Gartner Report
    • scalable and elastic IT-enabled capabilities
    • delivered as a service
  • Forrester Research
    • standardized IT capability
    • pay-per-use
    • self-service
  • NIST 2011 (adopted in this module)
    • a share pool of configureable computing resources
    • rapidly provisioned and released with minimal management effort

Benifits

  • current/future innovations and disruptions
  • advanced capability around AI, blockchain, IoT among others
  • reduces business cost: improves match between elastic resource demand and elastic computing resource
    • improves availability and elasticity
    • reduces cost
  • more economical cost model
    • traditional IT(on-premise)（预置）
      • fixed cost for equipment purchase and maintain
      • variable cost for using
    • cloud computing: only variable cost for using

Key Business Drivers: Elastic resource & Capacity planning

Capacity planning

  • Strategies
    • lead strategy: add capacity in anticipation of demand(prepare for future need)
    • lag strategy: add capacity when resources reaches its full capacity(enable more resources after demand overflow)
    • match strategy: add capacity in small increments as demand increases(always make sure the capacity is greater than demand)

Cost reduction

  • Costs: up-front invesment costs + operational costs
  • Operational costs
    • staffs
    • utility bills(power, cooling, …)
    • security
    • administration

Organization agility

  • elastic IT resources to response to business cycles beyond what was previously predicted or planned
  • software fixes/updates: updating datacenters is easier than updating millions of clients(centralize)
  • datacenter allows faster introduction of new hardware innovations(centralize)

Scaling

Basic Concepts and Terminology

  • Physical resources
    • Physical server
    • Storage device
    • Network device
  • Cloud resources
    • Virtual server
    • Software program
    • Service

Horizontal Scaling

  • same resource type

  • more nodes

  • Less expensive using commodity hardware

  • Resources instantly available

  • Resource replication and automated scaling

  • Additional IT resources needed

  • Not limited by hardware capacity

Vertical Scaling

  • replace resource with higher or lower capacity
  • add resources to a single node
  • More expensive using specialized hardware
  • Resources normally instantly available
  • May need additional setup
  • No additional IT resources needed
  • Limited by maximum hardware capacity

Trust Boundaries


Different organization subscribe same cloud service will meet overlapping of trust boundaries.

Cloud‐enabled Data Analytics

Big Data

  • Variety: Data types
    • Structured: Formal schema and data models
    • Unstructed: No pre-defined data model
    • Semi-structured: Lack strict data model structure
    • Mixed: Various types together
  • Velocity: Data production and processing speed
    • Batch: At time intervals
    • Near-time: At small time intervals
    • Real-time: Continuous input, process and output
    • Streams: Data flows
  • Volume: Data size
  • Veracity: Data reliability and trust
  • Value: Worth derived from exploiting big data

Analytics Workflow for Big Data

Analytics Solutions

  • Descriptive: model past behavior
  • Predictive: forecasts based on available data
  • Prescriptive: assess actions, assist decision making
